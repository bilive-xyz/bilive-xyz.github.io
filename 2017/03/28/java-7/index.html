Java反射机制

发表于 2017-03-28 | 分类于 java |

反射机制是什么

反射机制是在运行状态中，对于任意一个类，都能知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意一个方法和属性；这种动态获取的信息及动态调用对象的方法的功能称为java语言的反射机制。

通俗的说，反射机制就是把一个类，类的成员（函数，属性）当成一个对象操作。也就是说，类和类的成员，我们在幸运星时候还可以动态的去操作他它们。

反射机制能做什么

反射机制主要提供了以下功能：

  • 在运行时判断任意一个对象所属的类
  • 在运行时构造任意一个类的对象
  • 在运行时判断任意一个类所具有的成员变量和方法
  • 在运行时调用任意一个对象的方法
  • 生成动态代理

使用反射机制的一些地方

  • 工厂模式：Factory类中使用反射的话，添加了一个新的类之后，就不需要再修改工厂类Factory了

  • 分析类文件：能得到类中的方法等
  • 访问一些不能访问的变量或属性：破解别人代码
