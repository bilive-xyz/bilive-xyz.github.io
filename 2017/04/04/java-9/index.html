发表于 2017-04-04 | 分类于 mysql |

SQL语句分类

  • DDL(Date Definition Language)：数据库定义语言，用来定义数据库对象：库、表、列等。功能：创建、删除、修改库和表的结构。
  • DML(Data Manipulation Language)：数据库操作语言，用来定义数据库记录：增、删、改表记录。
  • DCL(Data Control Language)：数据库控制语言，用来定义访问权限和安全级别。
  • DQL(Data Query Language)：数据查询语言，用来查询记录。

DML（数据库操纵语言）

该语言用来对表记录操纵（增、删、改）。

插入数据：

说明：

  • 在数据库中所有的字符串类型，必须使用单引号。
  • （列名1，列名2，列名3）可省略，表示按照表的顺序插入。但不建议采取这种写法，因为降低了程序的可读性。

修改记录：

修改某列的全部值：update 表名 set 列名1=列值1(,列名2=列值2);
修改（某行或者多行记录的）列的指定值：update 表名 列名1=列值1 where 列名2=列值2 or 列名2=列值3;
运算符：=、!=、<>、<、>、>=、<=、between...and、in(…)、is null、not、or、and，其中in(…)的用法表示集合。例如：update 表名 set 列名1=列值1 where 列名2=列值2 or 列名2=列值3，用in(…)来写：update 表名 set 列名1=列值1 where 列名2 in (列值2,列值3)

删除数据：

不加where条件时会删除表中所有记录。

DCL（数据库控制语言）

该语言用来定义访问权限，不常用。

创建用户

用户只能在指定ip地址上登录mysql：create user 用户名@IP地址 identified by '密码';
用户可以在任意ip地址上登录：create user 用户名@'%' identified by '密码';

给用户授权

语法：grant 权限1,...,权限n on 数据库.* to 用户名@IP地址。其中权限1、2、n可以直接用all关键字代替。权限关键字例如：create,alter,drop,insert,update,delete,select。

撤销权限

语法：revoke 权限1，…，权限n on 数据库.* from 用户名@ ip地址；撤销指定用户在指定数据库上的指定权限。撤销例如：revoke create，delete on mydb1.* from user@localhost;表示撤销user用户在数据库mydb1上的create、alter权限。

查看用户权限

查看指定用户权限：show grants for 用户名@ip地址；

删除用户

drop user 用户名@ip地址；

DQL（数据库查询语言）

基本查询

字段（列）控制

  • 查询所有列:select * from 表名;其中＊表示查询所有列，而不是所有行的意思。
  • 查询指定列:select 列1，列2，列n from 表名；
  • 完全重复的记录只显示一次:在查询的列之前添加distinct:select distinct $ from 表名；缺省值为all。
  • 给列名起别名:select 列名1 (as) 别名1,列名2 (as) 别名2 from 表名；

条件控制

  • 条件查询。在后面添加where指定条件:select * from 表名 where 列名＝指定值;
  • 模糊查询：当你想查询所有姓张的记录。用到关键字like。eg:select * from 表名 where 列名 like ‘张’;(代表匹配任意一个字符，％代表匹配0～n个任意字符)。

排序

  • 升序：select * from 表名 order by 列名（ASC）;()里面的类容为缺省值
  • 降序:select * from 表名 order by 列名 DESC；
  • 使用多列作为排序条件：当第一列排序条件相同时，根据第二列条件排序，第二列条件相同视情况根据第三列排序。eg:select * from 表名 order by 列名1 ASC，列名2 DESC；意思是当列名1的值相同时按照列名2的值降序排。

分组查询

分组查询的信息都是组信息，不能查到个人的信息，其中查询组的信息是通过聚合函数得到的。

语法：select 分组列名，聚合函数1，聚合函数2 from 表名 group by 该分组列名其中分组列名需要的条件时该列名中有重复的信息。

查询的结果只能为：作为分组条件的列和聚合函数；查处的信息都是组的信息。

分组查询前，还可以通过关键字where先把满足条件的人分出来，再分组。语法为:select 分组列，聚合函数 from 表名 where 条件 group by 分组列；

分组查询后，也可以通过关键字having把组信息中满足条件的组再细分出来。语法为：select 分组列，聚合函数 from 表名 where 条件 group by 分组列 having 聚合函数或列名(条件)；

聚合函数

  • count:select count(列名) from 表名；,纪录有效行数。
  • max:select count(列名) from 表名；,列中最大值。
  • min:select sum(列名) from 表名；,列中最小值。
  • sum:select sum(列名) from 表名；,求列的总值，null 和字符串默认为0。
  • avg:select avg(列名) from 表名;,一列的平均值。

LIMIT子句(mysql中独有的语法)

LIMIT用来限定查询结果的起始行，以及总行数。
例如：select * from 表名 limit 4，3；表示起始行为第5行，一共查询3行记录。

学过的关键字：select,from,where,group by,having ,order by。当一条查询语句中都包含所有这些关键字时它们的优先级是select>from>where>group by>having>order by。
